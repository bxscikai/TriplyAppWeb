Triply

A simple trip tracker for global jetsetters

Features

Triply app is a simple travel tracking app that helps you keep track of your journeys and discover global festivals and destinations.

Key Features:

  • Google Calendar integration for seamless trip syncing
  • Pull up your trip details with a tap
  • Useful need to know information for each country
  • Visa checker tool to check if you need a visa to travel to destinations based on your passport
  • Worldwide festival discovery by country and month
  • Interactive passport to track visited countries
